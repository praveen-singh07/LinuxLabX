# 🚀 LinuxLabX

LinuxLabX is a modern Linux learning platform that provides an interactive terminal, hands-on labs, challenges, achievements, and progress tracking to help users master Linux in a gamified environment.

---

## ✨ Features

* 🔐 JWT Authentication System
* 🖥 Interactive Linux Terminal
* 📚 Hands-on Linux Labs
* 🎯 Real-world Challenges
* 🏆 Achievements & Badges
* 📈 Learning Analytics
* 🔥 XP & Streak System
* 🥇 Global Leaderboard
* 👤 User Profile & Settings
* 🌙 Modern Dark UI

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/LinuxLabX.git

cd LinuxLabX
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Server

```bash
uvicorn app.main:app --reload
```

Open:

```text
http://127.0.0.1:8000
```
